### Practical:  Steganography using cmd
### Definition:

Steganography is the technique of hiding secret data (like text, files, or images) inside another file ‚Äî such as an image, audio, or video ‚Äî so that no one can tell there‚Äôs hidden information.
Unlike encryption, which scrambles data, steganography hides its very existence.
Performing Steganography in Windows (Using CMD)
Here‚Äôs how you can hide and extract files using the Command Prompt (CMD) ‚Äî without any external tools.

‚Ä¢

o	A secret file you want to hide (example: secret.txt)
